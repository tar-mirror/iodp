NAME

IODP-FILE - input/output descriptor protocol for files

DESCRIPTION

This text documents the environment variables used to describes files or pipes.

ENVIRONMENT VARIABLES

IODP_n_PROTO
will be set to FILE.

IODP_n_LOCALNAME
will be set to the name of the file, if known.

SEE ALSO

The IODP specification iodp(5).
